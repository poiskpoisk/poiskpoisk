Регламент обслуживания газодизельного комплекта Аккорд.

Created Суббота 02 Сентябрь 2017

  1. Для проведения работ по обслуживанию газодизельных транспортных средств, необходимо иметь: Список оборудования и ПО для обслуживания
  2. Сервисная служба взаимодействует с службой эксплуатации ТС и выполняет работы по заявке служб эксплуатации транспортных средств. Заявка должна в обязательном порядке фиксироватся в Журнале регистрации заявок на обслуживание. Причиной обращения в Сервисную службу могут быть:
    • Плохой К замещения
    • Ненормальные звуки в двигателе
    • Перерасход газа
    • Пониженная тяга
    • Не включение газодизельного оборудования
    • Дерганье ТС
  3. Плановый осмотр и диагностика неисправностей транспортных средств, который следует проводить с интервалом не менее 3-х месяцев, даже при отсутствии заявок п.1. При плановом осмотре осуществляется Измерение К замещения, Проверка герметичности газовой системы, проверяется общее состояние оборудования. Результаты планового осмотра фиксируются в Журнал регистрации планового осмотра ТС.
  4. Сервисная служба взаимодействует с Газовой службой и передает ей заявки ( заявки на обслуживание Газовой службой регистрируются в Журнал регистрации заявок на обслуживание Газовой службой ) на:
    • Устранение утечек газа
    • Обеспечение нормативного давления газа
    • Замену вышедшего из строя газового оборудования ( исключая форсунки )
